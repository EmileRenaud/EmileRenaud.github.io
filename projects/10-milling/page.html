Milling Machine

CAD Milling

Project Description

While 3D printing is great, sometimes metal parts are necessary, which is why I want to build a milling machine that is small enough to fit in my apartment. This is a tough project because it is a 100% mechanical project. No electronics, no programming, just mechanics and a lot of precision.

I already made a small CNC in the past but this time, I prefer to make a conventional milling machine to be able to operate it faster without the trouble of programming a path for the CNC

Current Status

The frame is all welded together. Linear rail for the X axis are mounted onto the frame.
