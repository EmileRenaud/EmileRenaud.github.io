NEMA17 Close loop Controller

PCB 3D top view

Project Description

It is a "simple" stepper motor driver that you can screw onto a NEMA17.

Its main features:

  • Swapable A4899 style Stepper Driver. (if you burn one, you can change it!)
  • Current sensor to not overdrive the motor
  • Magnetic encoder for close loop feedback
  • ESP32-C3 for connectivity (Wifi,Bt,USB,UART)
  • Over and reverse voltage protection onboard

All the electrical files are final for the V1 here

The software is currently under development

Future Upgrade

  • USB-C connector with PD
  • Pinout for UART for smarter stepper driver like TMC
