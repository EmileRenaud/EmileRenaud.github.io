Motor Controller

Overview Machine

Project Description

I made a little motor controller for a friend's project. The main requirement was :

  • Smallest
  • Control one motor on/off
  • Variable speed
  • Can handle high current
  • Powered by 3S lipo

I used a small AtTiny85 because I can easily programme it in USB, small package and enough IO. A 7 amps continus mosfet is used to control the motor. And finally, a potentiometer to adjust the speed.
